# PHP+MySQL游戏信息管理系统 java528

## 项目概述

本项目是基于PHP和MySQL开发的游戏信息管理系统。系统包括前台用户模块和后台管理员模块，提供游戏信息浏览、搜索、评论以及用户注册、登录等功能。

## 技术栈

- 后端语言：PHP 5.6 及以上
- 数据库：MySQL
- 集成环境：phpstudy（推荐）、Wampsever、Xampp

## 功能模块

### 前台用户模块

- 用户注册与登录
- 浏览所有游戏信息与分类
- 搜索游戏
- 游戏详情页查看、收藏与评论
- 个人中心信息修改
- 收藏管理

### 管理员后台模块

- 数据统计详情查看
- 游戏信息增删改查
- 用户信息与评论管理

## 系统角色

- 普通用户：注册、登录、浏览、评论、收藏等
- 管理员：后台登录，系统管理与数据维护

## 运行环境

- PHP版本在5.6及以上

## 部署步骤

1. 准备集成环境（如phpstudy、Wampsever、Xampp）
2. 配置数据库（包含6张表）
3. 部署代码到服务器
4. 配置服务器以满足运行环境要求

```
├── frontend             // 前台代码目录

```
